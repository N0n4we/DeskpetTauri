# Deskpet APP

client stack: Tauri + Vue + TypeScript

server stack: Express.js

## Dev Setup

**Server**

```sh
cd backend
npm install
node index.js
```

**Client**

```sh
nix-shell --run 'pnpm tauri dev'
```
